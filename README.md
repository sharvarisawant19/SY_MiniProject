The Smart Surveillance and Guidance System was developed to address common challenges faced by visitors and students within a college campus. Many visitors struggle to locate classrooms, faculty members, and non-teaching staff. Additionally, students often remain unaware of classroom availability and important notices.
Technologies Used:
•	User Interface: Python (Tkinter)
•	Database: MySQL
Project Highlights:
•	Designed a user-friendly interface for easy navigation.
•	Provided accurate classroom navigation routes for all floors and wings.
•	Displayed real-time classroom occupancy status based on timetable data.
•	Enabled instant access to faculty and non-teaching staff contact information.
•	Integrated an efficient feedback management system with structured database storage.
This project enhanced my understanding of GUI development, database connectivity, and real-world problem-solving.
