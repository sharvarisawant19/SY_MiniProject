# Smart Surveillance and Guidance System

## Overview
The **Smart Surveillance and Guidance System** was developed to address common challenges faced by visitors and students within a college campus. Many visitors struggle to locate classrooms, faculty members, and non-teaching staff. Additionally, students often remain unaware of classroom availability and important notices.

This system provides an easy-to-use platform that helps users navigate the campus efficiently and access important information quickly.

---

## Technologies Used

- **User Interface:** Python (Tkinter)
- **Database:** MySQL

---

## Features / Project Highlights

- Designed a **user-friendly interface** for easy navigation.
- Provided **accurate classroom navigation routes** for all floors and wings.
- Displayed **real-time classroom occupancy status** based on timetable data.
- Enabled **instant access to faculty and non-teaching staff contact information**.
- Integrated an **efficient feedback management system** with structured database storage.

---

## Learning Outcomes

This project helped in gaining practical experience in:

- GUI development using **Tkinter**
- **Database connectivity** with MySQL
- **Backend data management**
- Solving **real-world campus navigation problems**

---

## Conclusion

The Smart Surveillance and Guidance System improves campus accessibility by providing clear directions, real-time classroom information, and easy access to staff contacts. It serves as a helpful tool for both visitors and students while demonstrating the practical application of Python and database integration.
